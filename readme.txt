                                                      使          用            说             明

软件开发环境：Python（版本: 3.9.19），开发时用到的库有NumPy（版本: 1.26.4）、Matplotlib（版本: 3.8.4）、ObsPy（版本: 1.4.1）、PyWavelets（版本: 1.6.0）等。

软件有两种形式：

（1）有GUI界面的交互软件：

第一种运行方式为：直接双击./DFSpy2025/DFSpy2025.exe文件打开软件，无需额外配置环境，只能windows环境下运行；

第二种运行方式为：运行源程序./dfspy_gui/main.py打开软件，需要配置python3.9以上的编程环境，并安装必要的第三方库（NumPy、Matplotlib、ObsPy、PyWavelets、scipy、PyQt5等），PyQt5的建议版本为5.15.11；

（2）无GUI界面的NOTEBOOK程序，支持windows/linux等环境（只在windows 11系统中测试过），需要配置python3.9以上的编程环境，并安装必要的第三方库（NumPy、Matplotlib、ObsPy、PyWavelets、scipy等）示例程序如DFSPy_test.ipynb。
